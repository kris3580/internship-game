# MobileMCP

description: Android emulator control via npx.
used_by: [[QA]]
install: `codex mcp add mobile-mcp npx "@mobilenext/mobile-mcp@latest"`

## Key Tools
| Tool | Action |
|---|---|
| `install_app` | Install APK on emulator |
| `launch_app` | Start the app |
| `take_screenshot` | Capture screen |
| `click_on_screen` | Tap at coordinates |
| `swipe` | Swipe gesture |
